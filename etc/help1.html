Frequently Asked Question

  • Q: Bagaimana cara / langkah yg harus saya lakukan agar dapat dieditkan?
  • A: Langkah-langkah yg bisa kamu lakukan, antara lain:
  • Silahkan menuju dan scroll down ke bagian contact me
  • Gunakan salah satu metode [ Whatsapp, Email, atau Telegram] untuk mengirimkan file skripsi anda yg ingin dieditkan
  • Kirim file skripsi (+ file tambahan bila ada*) dan sertakan keterangan tambahan bila ada (misalnya pilihan harga STANDARD atau LITE)
  • File yg saya terima akan saya cek terlebih dahulu untuk mengkonfirmasi bagian apa saja yg perlu diperbaiki
  • Setelah saya infokan hal2 yg bisa saya perbaiki, anda bisa pastikan kembali harga yg mau anda ambil
  • Setelah Deal, file akan saya kerjakan hari itu juga dan akan saya kabarkan bila file sudah selesai saya edit
  • Saya akan upload ke gdrive dan saya kirimkan link preview**) agar anda bisa mengecek hasil editan saya
  • Setelah anda yakin terkait editan tsb, anda bisa melakukan pembayaran melalui:
  • Setelah saya mengkonfirmasi pembayaran, maka link untuk file yg complete***) akan saya kirimkan kepada anda
  • setelah menerima file, diharap untuk mengecek kembali apabila ada bagian yg tidak sesuai, dan silahkan konfirmasi kesaya
  • Notes:
  • *) file tambahan misalnya adalah referensi mendeley, bila ingin editannya termasuk mengatur cara mengkutip dimendeley, atau file lain
  • **) link preview berisikan beberapa bagian yg diperbaiki dalam format pdf dan ppt terkait perubahan yg telah dilakukan agar dapat anda lihat dan cek sendiri
  • ***) file complete akan berupa format file word (.docx) sehingga nantinya anda masih dapat melakukan editan lagi
